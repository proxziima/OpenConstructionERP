# Animated Construction Background Package

Reusable animated background with estimation tables, dashboard charts, and construction blueprints.
Originally designed for OpenConstructionERP landing page.

## What's Inside

- `bg-styles.css` — All CSS for backgrounds, grid cells, lifecycle SVGs, animations
- `bg-animations.js` — JavaScript for animated table cells, chart updates, scroll effects
- `bg-demo.html` — Full working demo with all background types
- `README.md` — This file

## Quick Start

### 1. Include CSS & JS

```html
<link rel="stylesheet" href="bg-styles.css">
<!-- At end of body: -->
<script src="bg-animations.js"></script>
```

### 2. Add to any section

```html
<section style="position:relative; overflow:visible; padding:80px 0;">

  <!-- Animated estimation table (left side) -->
  <div class="oce-bg-table" data-side="left" data-rows="8">
    <!-- Auto-populated by JS -->
  </div>

  <!-- Dashboard chart (right side) -->
  <div class="oce-bg-chart" data-side="right" data-type="bars">
    <!-- SVG auto-generated by JS -->
  </div>

  <!-- Your content here -->
  <div style="position:relative; z-index:1;">
    <h2>Your Section Title</h2>
    <p>Your content...</p>
  </div>

</section>
```

### 3. Available background types

#### Animated Estimation Table
```html
<div class="oce-bg-table" data-side="left" data-rows="10"></div>
```
- `data-side`: "left" or "right" — which side to position
- `data-rows`: number of rows (6-14)
- Cells animate with construction data (positions, descriptions, quantities, prices)

#### Dashboard Charts
```html
<div class="oce-bg-chart" data-side="right" data-type="bars"></div>
```
Types: `bars`, `donut`, `sparkline`, `area`, `mixed`

#### Construction Blueprint (Lifecycle SVG)
```html
<div class="oce-bg-blueprint" data-side="right" data-stage="frame"></div>
```
Stages: `plot`, `frame`, `walls`, `interior`, `drawing`, `skyline`

### 4. Customization

#### Colors
Override CSS variables:
```css
:root {
  --oce-bg-color: #0066ff;      /* Primary blue */
  --oce-bg-opacity: 0.08;       /* Background opacity */
  --oce-bg-cell-active: 0.12;   /* Active cell opacity */
  --oce-bg-perspective: 14deg;  /* 3D tilt angle */
}
```

#### Opacity
Default is 8% for lifecycle SVGs, 4-15% for table cells. Adjust with CSS:
```css
.oce-bg-table { opacity: 0.06; }  /* More subtle */
.oce-bg-blueprint { opacity: 0.12; }  /* More visible */
```

#### Animation Speed
```css
.oce-bg-cell { transition: background 0.7s, color 0.5s; }  /* Cell fade speed */
```
```javascript
// In bg-animations.js, change interval:
OCE_BG_CONFIG.cellInterval = 500;  // ms between cell updates
OCE_BG_CONFIG.barInterval = 800;   // ms between bar jitters
```

#### Position & Size
```css
.oce-bg-table[data-side="left"] {
  left: 10px;       /* Distance from edge */
  width: 500px;     /* Table width */
  top: -10px;       /* Vertical offset */
}
```

### 5. Performance Notes

- Uses CSS transitions only (no requestAnimationFrame loops)
- `will-change: transform` for GPU compositing
- `pointer-events: none` — doesn't interfere with content
- `setInterval` at 500-800ms — very lightweight
- All SVGs inline — no external requests
- Hidden on mobile via `@media (max-width: 768px)`

### 6. Browser Support

- Chrome 80+, Firefox 80+, Safari 14+, Edge 80+
- CSS `mask-image` for gradient fade (WebKit prefix included)
- CSS `perspective` + `rotateX` for 3D tilt
