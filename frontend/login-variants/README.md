# Login Page Variants — Pick One

Three standalone mockups of a redesigned `/login` page. **Not for publication.**

Each file is self-contained (Tailwind via CDN) — open in a browser to preview:

| Variant | Style | Mood |
|---|---|---|
| `variant_1_glass_aurora.html` | Glassmorphism on a colorful aurora background | Premium SaaS, soft, optimistic |
| `variant_2_split_hero_video.html` | Split layout: hero copy + YouTube demo thumbnail on left, clean form on right | Marketing-driven, conversion-focused |
| `variant_3_dark_terminal.html` | Dark theme, monospace accents, neon glow | Developer-focused, "open-source flag-up-front" |

All three include:
- Prominent **language switcher** (top of the page, always visible)
- Direct **GitHub** link + repo star count placeholder
- **YouTube demo video** link/thumbnail
- Email/password fields + demo-account quick-login row
- "Remember me" + Forgot password
- Privacy/Terms removed (broken on prod, per Artem)

Pick one and tell me which → I'll port it into `LoginPage.tsx` and ship.
